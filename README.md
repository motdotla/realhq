# Realhq

A gem for my application to RealHQ.

## Installation

Add this line to your application's Gemfile:

    gem 'realhq'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install realhq

## Usage

rake realhq:scottmotte
